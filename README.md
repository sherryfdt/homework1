# 個人網頁專案

一個現代風格的響應式個人網頁，使用 HTML、CSS 和 TypeScript 開發，包含豐富的動畫效果。

## 📋 專案簡介

這是一個單頁面個人網頁，展示個人資訊、照片和聯絡方式。網頁採用現代設計風格，具有流暢的動畫效果和完整的響應式設計，可在各種裝置上完美顯示。

## ✨ 主要功能

### 1. 導覽列
- 固定頂部導覽列
- 滾動時自動改變樣式
- 平滑滾動導航
- 響應式設計

### 2. 英雄區塊
- 全螢幕歡迎區
- 漸層背景
- 載入動畫效果

### 3. 自我介紹區塊
- 個人簡介
- 技能專長展示
- 滾動觸發動畫

### 4. 照片展示區塊
- 三張照片展示
- **滑鼠懸停左右晃動動畫**（使用 TypeScript 控制）
- 卡片式設計
- 響應式佈局

### 5. 聯絡方式區塊
- 多種聯絡方式展示
- 懸停動畫效果
- 深色主題設計

## 🛠️ 技術棧

- **HTML5** - 網頁結構
- **CSS3** - 樣式設計（內嵌在 HTML 中）
  - Flexbox & Grid 佈局
  - CSS 動畫與過渡效果
  - 響應式設計（Media Queries）
- **TypeScript** - 動畫控制邏輯
- **JavaScript** - 編譯後的動畫腳本

## 🚀 使用方法

### 方法 1：直接開啟（最簡單）
1. 在檔案管理器中找到 `index.html`
2. 雙擊檔案，或右鍵選擇「用瀏覽器開啟」

### 方法 2：使用本地伺服器（推薦）
```bash
# 進入專案目錄
cd /home/sherry/task3_test/homework1

# 使用 Python 啟動本地伺服器
python3 -m http.server 8000

# 在瀏覽器中訪問
# http://localhost:8000
```

### 方法 3：使用 Node.js 伺服器
```bash
# 安裝 http-server（如果尚未安裝）
npm install -g http-server

# 啟動伺服器
http-server -p 8000

# 在瀏覽器中訪問
# http://localhost:8000
```

## 🎨 動畫效果說明

### 照片晃動動畫
- **觸發方式**：滑鼠移到照片上
- **動畫效果**：照片會左右來回晃動
- **停止方式**：滑鼠離開照片時自動停止
- **實現方式**：使用 TypeScript 控制 CSS 動畫類別的添加與移除

### 其他動畫
- **頁面載入**：英雄區塊淡入動畫
- **滾動觸發**：各區塊進入視窗時的淡入與滑入動畫
- **導覽列**：滾動時背景與陰影變化
- **懸停效果**：聯絡項目懸停時的上升動畫

## 🔧 自訂設定

### 更換照片
1. 將你的照片檔案放在專案目錄中
2. 在 `index.html` 中找到照片區塊（約第 424-432 行）
3. 修改 `src` 屬性為你的照片檔名：
```html
<img src="你的照片檔名.jpg" alt="照片 1">
```

### 修改內容
- **個人資訊**：編輯 `index.html` 中「關於我」區塊的內容
- **聯絡方式**：修改「聯絡方式」區塊中的資訊
- **樣式**：調整 `<style>` 標籤內的 CSS

### 調整動畫
如需修改動畫效果，可以：
1. 編輯 `animations.ts` 檔案
2. 使用 TypeScript 編譯器編譯：
```bash
tsc animations.ts
```
3. 或直接編輯 `animations.js` 檔案
